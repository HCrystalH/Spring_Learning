## 3. Lớp Generic cho Service (Business Layer)

### Interface: `GenericService.java`

```
package com.example.studentmgmt.service.base;

import java.util.List;
import java.util.Optional;

public interface GenericService<T, ID>{
    List<T> findAll();
    Optional<T> findById(ID id);
    T save(T entity);
    void deleteById(ID id);
}
```

### Implement: `GenericServiceImpl.java`

```
package com.example.studentmgmt.service.base;

import org.springframework.data.jpa.repository.JpaRepository;
import java.util.List;
import java.util.Optional;

public abstract class GenericServiceImpl<T, ID> implements GenericService<T, ID>{

    protected final JpaRepository<T, ID> repository;

    protected GenericServiceImpl(JpaRepository<T, ID> repository){
        this.repository = repository;
    }

    @Override
    public List<T> findAll(){
        return repository.findAll();
    }

    @Override
    public Optional<T> findById(ID id){
        return repository.findById(id);
    }

    @Override
    public T save(T entity){
        return repository.save(entity);
    }

    @Override
    public void deleteById(ID id){
        repository.deleteById(id);
    }
}
```

## 4. Lớp Generic cho Controller (Web Layer)

Để viết Generic cho Controller hiển thị Thymeleaf, điểm mấu chốt là cần cấu hình linh hoạt các **View Folder** và tên **Model Attribute**.

### Base Controller: `GenericController.java`

```
package com.example.studentmgmt.controller.base;

import com.example.studentmgmt.service.base.GenericService;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.*;

public abstract class GenericController<T, ID>{

    protected final GenericService<T, ID> service;
    protected final String viewFolder;       // Thư mục chứa view Thymeleaf (VD: "students")
    protected final String modelName;        // Tên biến dùng trong form/list (VD: "student")

    protected GenericController(GenericService<T, ID> service, String viewFolder, String modelName){
        this.service = service;
        this.viewFolder = viewFolder;
        this.modelName = modelName;
    }

    @GetMapping
    public String list(Model model){
        model.addAttribute(modelName + "s", service.findAll());
        return viewFolder + "/list";
    }

    @GetMapping("/create")
    public String createForm(Model model) throws Exception{
        // Tự động khởi tạo instance mới cho Model (Yêu cầu Class có No-Args Constructor)
        model.addAttribute(modelName, getEntityClass().getDeclaredConstructor().newInstance());
        return viewFolder + "/form";
    }

    @GetMapping("/edit/{id}")
    public String editForm(@PathVariable ID id, Model model){
        service.findById(id).ifPresent(entity -> model.addAttribute(modelName, entity));
        return viewFolder + "/form";
    }

    @PostMapping("/save")
    public String save(@ModelAttribute T entity){
        service.save(entity);
        return "redirect:/" + viewFolder;
    }

    @GetMapping("/delete/{id}")
    public String delete(@PathVariable ID id){
        service.deleteById(id);
        return "redirect:/" + viewFolder;
    }

    // Phương thức bổ trợ để lấy Class type của Entity phục vụ tạo mới instance
    protected abstract Class<T> getEntityClass();
}
```

### Triển khai cụ thể: `StudentController.java`

Nhờ có Generic lớp cha, Controller của Student cực kỳ ngắn gọn:

```
package com.example.studentmgmt.controller;

import com.example.studentmgmt.controller.base.GenericController;
import com.example.studentmgmt.model.Student;
import com.example.studentmgmt.service.StudentService;
import org.springframework.stereotype.Controller;
import org.springframework.web.bind.annotation.RequestMapping;

@Controller
@RequestMapping("/students")
public class StudentController extends GenericController<Student, Long>{

    public StudentController(StudentService service){
        // Định nghĩa: thư mục view là "students", tên attribute là "student"
        super(service, "students", "student");
    }

    @Override
    protected Class<Student> getEntityClass(){
        return Student.class;
    }
}
```

## 5. View với Thymeleaf

Tạo các file giao diện trong thư mục `src/main/resources/templates/students/`.

### Danh sách sinh viên: `list.html`

```
<!DOCTYPE html>
<html xmlns:th="http://www.thymeleaf.org">
<head>
    <title>Danh sách Sinh viên</title>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
</head>
<body class="container mt-5">
    <h2>Quản lý Sinh viên</h2>
    <a th:href="@{/students/create}" class="btn btn-primary mb-3">Thêm sinh viên mới</a>

    <table class="table table-bordered">
        <thead>
            <tr>
                <th>ID</th>
                <th>Họ ên</th>
                <th>Email</th>
                <th>Chuyên ngành</th>
                <th>Hành động</th>
            </tr>
        </thead>
        <tbody>
            <tr th:each="st : ${students}">
                <td th:text="${st.id}"></td>
                <td th:text="${st.name}"></td>
                <td th:text="${st.email}"></td>
                <td th:text="${st.major}"></td>
                <td>
                    <a th:href="@{/students/edit/{id}(id=${st.id})}" class="btn btn-warning btn-sm">Sửa</a>
                    <a th:href="@{/students/delete/{id}(id=${st.id})}" class="btn btn-danger btn-sm"
                       onclick="return confirm('Bạn có chắc chắn muốn xóa?')">Xóa</a>
                </td>
            </tr>
        </tbody>
    </table>
</body>
</html>
```

Cấu hình cho application

```jsx
# ===================================================================
# H2 DATABASE CONFIGURATION (LONG-TERM FILE-BASED)
# ===================================================================

# 1. Đường dẫn kết nối tới File thay vì Memory
# Chú ý: Dữ liệu sẽ được lưu vào thư mục "./data/studentdb.mv.db" trong project của bạn
spring.datasource.url=jdbc:h2:file:./data/studentdb;DB_CLOSE_DELAY=-1;DB_CLOSE_ON_EXIT=FALSE

spring.datasource.driverClassName=org.h2.Driver
spring.datasource.username=sa
spring.datasource.password=password

# 2. Bật H2 Console để quản lý dữ liệu qua trình duyệt
spring.h2.console.enabled=true
spring.h2.console.path=/h2-console
# Cho phép truy cập từ các ứng dụng khác hoặc hiển thị trong thẻ iframe nếu cần
spring.h2.console.settings.web-allow-others=true

# 3. Cấu hình Hibernate cho việc lưu trữ lâu dài
# "update": Tự động cập nhật cấu trúc bảng cấu hình từ Entity mà không làm mất dữ liệu cũ
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
spring.jpa.properties.hibernate.format_sql=true
spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.H2Dialect
```

## 6. Trải nghiệm kết quả

1. Thêm dependency cần thiết vào `pom.xml`: `Spring Web`, `Spring Data JPA`, `Thymeleaf`, `Lombok`, và DB Driver (H2 hoặc MySQL).
2. Chạy ứng dụng.
3. Truy cập: `http://localhost:8080/students`để kiểm tra toàn bộ luồng CRUD.

> **Lưu ý về thiết kế:** Việc viết Generic Controller kết hợp Thymeleaf yêu cầu tuân thủ nghiêm ngặt quy tắc đặt tên thư mục views (`/templates/{viewFolder}/list.html`và `form.html`) để lớp cha có thể map tự động một cách chính xác.
